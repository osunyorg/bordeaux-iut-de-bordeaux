Les demandes de dossier de candidature en licence professionnelle à l’IUT de Bordeaux se font uniquement par internet via le site CandIUT. Ce site est accessible à partir du 1ᵉʳ février. Le calendrier de recrutement de chaque licence professionnelle est visible sur CandIUT.

notes: >- alt: >- credit: >- - kind: heading title: >- Formation continue et alternance slug: "formation-continue-et-alternance" position: 2 rank: 2 - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

Vous êtes demandeur d’emploi, salarié, auto-entrepreneur… et vous souhaitez reprendre vos études, le service FCA est là pour vous renseigner sur les modalités. Attention, certains financements nécessitent un délai d’instruction long et imposé par les organismes financeurs (projet de transition professionnelle par exemple). Ainsi, merci de contacter le service FCA avant de candidater (idéalement dès février pour une rentrée en septembre). Nous pourrons ainsi envisager avec vous toutes les possibilités de financement.
