Si vous n’avez jamais été inscrit(e) à l’université de Bordeaux, les inscriptions seront ouvertes à partir du 16 juillet 2024. Vous pouvez suivre la procédure d’inscription selon votre type d’admission.

Si vous avez déjà été inscrit(e) à l’université de Bordeaux et que vous possédez un n°étudiant, les inscriptions seront ouvertes à partir du 16 juillet 2024. Vous pouvez suivre la procédure d’inscription selon votre type d’admission.

notes: >- alt: >- credit: >- - kind: block template: files title: >- Documents disponibles en téléchargement slug: >- documents-disponibles-en-telechargement ranks: self: 3 data: description: >-

Retrouvez ci-dessous tous les documents nécessaires pour effectuer votre inscription administrative en fonction de votre profil. Selon la procédure qui vous concerne, un ou plusieurs de ces documents pourront vous être demandés. Vous pouvez les télécharger librement.

Consultez la procédure d'inscription correspondant à votre profil en cliquant sur l'un des menus déroulants ci-dessous.

1. Se connecter à cvec.etudiant.gouv.fr - lien externe pour régler la CVEC (Contribution vie étudiante et de campus)* et obtenir un n° d’attestation

- title: >- text: >-

2. Se connecter à IAWEB Primo - lien externe pour régler les droits d’inscription**

N° de candidature : 24IUT + n° de candidature CANDIUT - lien externe (à ne pas confondre avec le numéro de candidat).

- title: >- text: >-

3. Se connecter à PJWEB - lien externe et déposer les documents demandés

Pour en savoir plus sur les documents demandés, vous pouvez consulter la note de précisions disponible en téléchargement.

- title: >- text: >-

4. Activer son identité numérique IDNUM - lien externe pour se connecter à l’ENT - lien externe (Espace numérique de travail)

- kind: block template: chapter title: >- slug: >- ranks: data: layout: no_background text: >-

* Les étudiants boursiers doivent se connecter, mais ne payent pas la CVEC.

** Les étudiants boursiers ne payent pas les droits d’inscription.

Document à télécharger :

  • Autorisation d’utilisation des données personnelles et de la photographie

Quand l’ensemble de vos pièces justificatives sera validé, vous pourrez :

  • Télécharger votre certificat de scolarité sur l’ENT (onglet Mon cursus > Mes documents administratifs)
  • Récupérer votre carte étudiante le jour de la rentrée ou au service Formation et vie universitaire (FVU)
